# Reglas de Filtrado

Repositorio con la documentacion operativa de filtrado para `solicitudes-app`.

## Archivos

- `FILTRADO-AUTONOMO.md`
  - guia minima para ejecutar el filtrado autonomo
- `matriz-validacion-filtrado-v2.md`
  - matriz operativa de checks y fuentes de validacion
- `regla-complementaria-creditos-avales.md`
  - regla complementaria para creditos a avales

## Alcance

Este repo solo concentra reglas de filtrado. No incluye codigo del backend ni del frontend.
